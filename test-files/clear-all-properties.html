üîß Database Clear Options

Option 1: Clear via Owner Dashboard (Recommended)

  1. Go to Owner Dashboard: Open Dashboard
  2. For each property, click the "Delete" button
  3. Confirm deletion for each property
  4. Repeat until all properties are removed

Option 2: Clear via Database Console

If you have access to Supabase dashboard:

  1. Go to Supabase Dashboard
  2. Navigate to Table Editor
  3. Select "properties" table
  4. Delete all rows

Option 3: SQL Command

Run this SQL command in Supabase SQL Editor:

DELETE FROM properties;
