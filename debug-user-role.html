Step 4: Database Check

If the user role is showing as 'customer', the issue is in the database. The user's role in the database needs to be updated to 'owner'.

Quick Fix: If you have access to the database, run this SQL:

-- Update user role to owner (replace 'your-email@example.com' with actual email) UPDATE users SET role = 'owner' WHERE email = 'your-email@example.com';
