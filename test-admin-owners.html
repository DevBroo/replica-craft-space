üìä Database Integration

Data Source

  • Table: profiles
  • Filter: role = 'property_owner'
  • Properties Count: Joined with properties table
  • Ordering: Most recent registrations first

API Endpoints Used

GET /rest/v1/profiles?role=eq.property_owner
GET /rest/v1/properties?owner_id=eq.{id}&select=*&count=exact
PUT /rest/v1/profiles?id=eq.{id}&role=eq.property_owner
