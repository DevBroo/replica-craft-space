🔧 Agent Booking Fix

🐛 Issue Identified

❌ Problem:

Booking Restricted Error: "Only customers can book properties"

Location: PropertyDetails.tsx line 550-557

Impact: Agents couldn't book properties for customers

✅ Fix Applied

🎯 Changes Made:

  • PropertyDetails.tsx - Updated role check to allow agents
  • BookingPayment.tsx - Added agent_id to booking request
  • PaymentSuccess.tsx - Added agent_id to booking creation
  • Automatic Commission Tracking - Agent bookings now tracked for commission

📊 Before vs After

❌ Before:

if (user.role !== "customer") { toast({ title: "Booking Restricted", description: "Only customers can book properties.", variant: "destructive", }); return; }

Result: Agents blocked from booking

✅ After:

if (user.role !== "customer" && user.role !== "agent") { toast({ title: "Booking Restricted", description: "Only customers and agents can book properties.", variant: "destructive", }); return; }

Result: Agents can now book properties
